Accessibility Statement

Last updated: January 1, 2025

Our Commitment

Pacific Sands is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Accessibility Features

Our website includes the following accessibility features:

  • Semantic HTML structure for screen readers
  • Alt text for all images
  • Keyboard navigation support
  • High contrast color schemes
  • Responsive design for all devices
  • Clear heading structure
  • Focus indicators for keyboard users

Standards

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 level AA standards.

Feedback

Assistive Technology

Our website is designed to work with assistive technologies including:

  • Screen readers
  • Voice recognition software
  • Switch navigation devices
  • Magnification software
