Declaración de Accesibilidad

Última actualización: 1 de enero de 2025

Nuestro Compromiso

Pacific Sands está comprometido a asegurar la accesibilidad digital para personas con discapacidades. Estamos mejorando continuamente la experiencia del usuario para todos y aplicando los estándares de accesibilidad relevantes.

Características de Accesibilidad

Nuestro sitio web incluye las siguientes características de accesibilidad:

  • Estructura HTML semántica para lectores de pantalla
  • Texto alternativo para todas las imágenes
  • Soporte de navegación por teclado
  • Esquemas de color de alto contraste
  • Diseño responsivo para todos los dispositivos
  • Estructura de encabezados clara
  • Indicadores de enfoque para usuarios de teclado

Estándares

Nuestro objetivo es cumplir con los estándares de las Pautas de Accesibilidad de Contenido Web (WCAG) 2.1 nivel AA.

Tecnología de Asistencia

Nuestro sitio web está diseñado para funcionar con tecnologías de asistencia incluyendo:

  • Lectores de pantalla
  • Software de reconocimiento de voz
  • Dispositivos de navegación por interruptores
  • Software de magnificación
